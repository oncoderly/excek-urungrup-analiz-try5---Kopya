=====================================================
Excel Analiz Uygulamasi - Portable Surum
=====================================================

Bu uygulama, Excel dosyalarini analiz etmek icin tasarlanmis portable bir Streamlit uygulamasidir.

SISTEM GEREKSINIMLERI:
=====================================================
- Windows 7/10/11 (64-bit)
- Python 3.8+ (python.exe PATH'de olmali)
- Internet baglantisi (ilk kurulum icin)

KURULUM ADIMLARI:
=====================================================
1. Tum dosyalari bir klasore cikarin
2. "setup_portable.bat" dosyasini yonetici olarak calistirin
3. Kurulum tamamlanana kadar bekleyin

KULLANIM:
=====================================================
1. "run_app.bat" dosyasini cift tiklayin
2. Tarayicinizda uygulama otomatik olarak acilacak
3. Excel dosyanizi yukleyin ve analiz edin
4. Uygulamayi kapatmak icin komut penceresini kapatin

DOSYA YAPISI:
=====================================================
- excel_analyzer.py         : Ana uygulama
- requirements.txt          : Gerekli kutuphaneler
- setup_portable.bat        : Kurulum scripti
- run_app.bat              : Uygulama baslatma scripti
- venv/                    : Python sanal ortami (kurulumdan sonra)

SORUN GIDERME:
=====================================================
- Python bulunamadi hatasi: Python'un yuklu oldugunu ve PATH'e eklendigini kontrol edin
- Kutuphaneler yuklenemedi: Internet baglantinizi kontrol edin
- Uygulama acilmiyor: Antivirus yaziliminin engelleyip engellemedigini kontrol edin

NOTLAR:
=====================================================
- Ilk kurulum internet baglantisi gerektirir
- Kurulum tamamlandiktan sonra offline calisabilir
- Sanal ortam sayesinde sistem Python'u etkilenmez
- Uygulamayi baska bir bilgisayara kopyalayarak tasiyabilirsiniz
